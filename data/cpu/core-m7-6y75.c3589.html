The Intel Core m7-6Y75 was a mobile processor with 2 cores, launched in September 2015, at an MSRP of $393. It is part of the Core m7 lineup, using the Skylake-Y architecture with BGA 1515. Thanks to Intel Hyper-Threading the core-count is effectively doubled, to 4 threads. Core m7-6Y75 has 4 MB of L3 cache and operates at 1200 MHz by default, but can boost up to 3.1 GHz, depending on the workload. Intel is building the Core m7-6Y75 on a 14 nm production process, the transistor count is unknown. The multiplier is locked on Core m7-6Y75, which limits its overclocking capabilities.
With a TDP of 5 W, the Core m7-6Y75 consumes extremely little energy. Intel's processor supports DDR3 memory with a dual-channel interface. The highest officially supported memory speed is 1866 MT/s, but with overclocking (and the right memory modules) you can go even higher. For communication with other components in the system, Core m7-6Y75 uses a PCI-Express Gen 3 connection. This processor features the Intel HD 515 integrated graphics solution.
Hardware virtualization is available on the Core m7-6Y75, which greatly improves virtual machine performance. Additionally, IOMMU virtualization (PCI passthrough) is supported, so that guest virtual machines may directly use host hardware. Programs using Advanced Vector Extensions (AVX) will run on this processor, boosting performance for calculation-heavy applications. Besides AVX, Intel is including the newer AVX2 standard, too, but not AVX-512.
