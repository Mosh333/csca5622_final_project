The Intel Pentium G3420 is a desktop processor with 2 cores, launched in September 2013. It is part of the Pentium lineup, using the Haswell architecture with Socket 1150. Pentium G3420 has 3 MB of L3 cache and operates at 3.2 GHz. Intel is making the Pentium G3420 on a 22 nm production node using 1,400 million transistors. The multiplier is locked on Pentium G3420, which limits its overclocking potential.
With a TDP of 54 W, the Pentium G3420 consumes typical power levels for a modern PC. Intel's processor supports DDR3 memory with a dual-channel interface. For communication with other components in the machine, Pentium G3420 uses a PCI-Express Gen 3 connection. This processor features the Intel HD (Haswell) integrated graphics solution.
Hardware virtualization is available on the Pentium G3420, which greatly improves virtual machine performance.
