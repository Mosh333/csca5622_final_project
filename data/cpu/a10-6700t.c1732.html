The AMD A10-6700T was a desktop processor with 4 cores, launched in September 2013. It is part of the A10 lineup, using the Richland architecture with Socket FM2. A10-6700T has 4 MB of L2 cache and operates at 2.5 GHz by default, but can boost up to 3.5 GHz, depending on the workload. AMD is making the A10-6700T on a 32 nm production node using 1,303 million transistors. The silicon die of the chip is not fabricated at AMD, but at the foundry of GlobalFoundries. The multiplier is locked on A10-6700T, which limits its overclocking potential.
With a TDP of 45 W, the A10-6700T consumes typical power levels for a modern PC. AMD's processor supports DDR3 memory with a dual-channel interface. The highest officially supported memory speed is 1866 MT/s, but with overclocking (and the right memory modules) you can go even higher. For communication with other components in the system, A10-6700T uses a PCI-Express Gen 2 connection. This processor features the Radeon HD 8650D integrated graphics solution.
Hardware virtualization is available on the A10-6700T, which greatly improves virtual machine performance. Programs using Advanced Vector Extensions (AVX) can run on this processor, boosting performance for calculation-heavy applications.
