The Intel Xeon E7-8850 was a server/workstation processor with 10 cores, launched in April 2011. It is part of the Xeon E7 lineup, using the Westmere-EX architecture with Socket 1567. Thanks to Intel Hyper-Threading the core-count is effectively doubled, to 20 threads. To further increase overall system performance, up to eight Xeon E7-8850 CPUs can work together in a multi-processor (SMP) configuration. Xeon E7-8850 has 30 MB of L3 cache and operates at 2000 MHz by default, but can boost up to 2.4 GHz, depending on the workload. Intel is making the Xeon E7-8850 on a 32 nm production node using 2,600 million transistors. The multiplier is locked on Xeon E7-8850, which limits its overclocking potential.
With a TDP of 130 W, the Xeon E7-8850 consumes a lot of power, so good cooling is definitely needed. Intel's processor supports DDR3 memory. ECC memory is supported, too, which is an important capability for mission-critical systems, to avoid data corruption. This processor lacks integrated graphics, you might need a graphics card.
Hardware virtualization is available on the Xeon E7-8850, which greatly improves virtual machine performance. Additionally, IOMMU virtualization (PCI passthrough) is supported, so that guest virtual machines may directly use host hardware.
