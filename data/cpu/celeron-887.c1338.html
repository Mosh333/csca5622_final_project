The Intel Celeron 887 is a mobile processor with 2 cores, launched in September 2012. It is part of the Celeron lineup, using the Sandy Bridge architecture with BGA 1023. Celeron 887 has 2 MB of L3 cache and operates at 1500 MHz. Intel is making the Celeron 887 on a 32 nm production node using 504 million transistors. The multiplier is locked on Celeron 887, which limits its overclocking potential.
With a TDP of 17 W, the Celeron 887 consumes very little energy. Intel's processor supports DDR3 memory with a dual-channel interface. This processor features the Intel HD (Sandy Bridge) integrated graphics solution.
Hardware virtualization is available on the Celeron 887, which greatly improves virtual machine performance.
