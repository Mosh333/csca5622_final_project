NVIDIA GV10B

GV10B
Die Shot
SM Diagram
NVIDIA's GV10B GPU uses the Volta architecture and is made using a 12 nm production process at TSMC. With a die size of 350 mm² and a transistor count of 9,000 million it is a large chip. GV10B supports DirectX 12 (Feature Level 12_1). For GPU compute applications, OpenCL version 1.2 and CUDA 7.2 can be used. It features 512 shading units, 32 texture mapping units and 16 ROPs. Also included are 64 tensor cores which help improve the speed of machine learning applications.
