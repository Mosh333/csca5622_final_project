AMD Redwood

Redwood
Block Diagram
AMD's Redwood GPU uses the TeraScale 2 architecture and is made using a 40 nm production process at TSMC. With a die size of 104 mm² and a transistor count of 627 million it is a small chip. Redwood supports DirectX 11.2 (Feature Level 11_0). For GPU compute applications, OpenCL version 1.2 can be used. It features 400 shading units, 20 texture mapping units and 8 ROPs.
