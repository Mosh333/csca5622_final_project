The GeForce GTX 760 OEM was a graphics card by NVIDIA, launched on November 6th, 2016. Built on the 28 nm process, and based on the GK104 graphics processor, the card supports DirectX 12. Even though it supports DirectX 12, the feature level is only 11_0, which can be problematic with newer DirectX 12 titles. The GK104 graphics processor is an average sized chip with a die area of 294 mm² and 3,540 million transistors. Unlike the fully unlocked GeForce GTX 680, which uses the same GPU but has all 1536 shaders enabled, NVIDIA has disabled some shading units on the GeForce GTX 760 OEM to reach the product's target shader count. It features 1344 shading units, 112 texture mapping units, and 32 ROPs. NVIDIA has paired 2,048 MB GDDR5 memory with the GeForce GTX 760 OEM, which are connected using a 256-bit memory interface. The GPU is operating at a frequency of 993 MHz, which can be boosted up to 1046 MHz, memory is running at 1650 MHz (6.6 Gbps effective).
Being a dual-slot card, the NVIDIA GeForce GTX 760 OEM draws power from 2x 6-pin power connectors, with power draw rated at 170 W maximum. Display outputs include: 2x DVI, 1x HDMI 1.4a, 1x DisplayPort 1.2. GeForce GTX 760 OEM is connected to the rest of the system using a PCI-Express 3.0 x16 interface. The card measures 241 mm in length, and features a dual-slot cooling solution.
