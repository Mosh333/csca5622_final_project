AMD Scrapper

AMD's Scrapper GPU uses the TeraScale 3 architecture and is made using a 32 nm production process at GlobalFoundries. With a die size of 246 mm² and a transistor count of 1,303 million it is a medium-sized chip. Scrapper supports DirectX 11.2 (Feature Level 11_0). For GPU compute applications, OpenCL version 1.2 can be used. It features 384 shading units, 24 texture mapping units and 8 ROPs.
