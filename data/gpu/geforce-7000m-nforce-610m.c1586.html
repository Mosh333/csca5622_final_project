The GeForce 7000M + nForce 610M was a mobile integrated graphics solution by NVIDIA, launched on February 1st, 2006. Built on the 90 nm process, and based on the C67 graphics processor, in its MCP67MV-A2 variant, the device supports DirectX 9.0c. Since GeForce 7000M + nForce 610M does not support DirectX 11 or DirectX 12, it might not be able to run all the latest games. It features 2 pixel shaders and 1 vertex shader 1 texture mapping unit and 1 ROP. Due to the lack of unified shaders you will not be able to run recent games at all (which require unified shader/DX10+ support). The GPU is operating at a frequency of 350 MHz.
Its power draw is not exactly known. GeForce 7000M + nForce 610M is connected to the rest of the system using a PCI-Express 1.0 x16 interface.
