AMD Tobago

Tobago
Tobago PRO
Block Diagram
AMD's Tobago GPU uses the GCN 2.0 architecture and is made using a 28 nm production process at TSMC. With a die size of 160 mm² and a transistor count of 2,080 million it is a small chip. Tobago supports DirectX 12 (Feature Level 12_0). For GPU compute applications, OpenCL version 2.1 can be used. It features 896 shading units, 56 texture mapping units and 16 ROPs.
