NVIDIA GK104

GK104
GK104-850-A2
GK104 BGA
Oleg Kashirin
Die Shot
Block Diagram
SMX Diagram
NVIDIA's GK104 GPU uses the Kepler architecture and is made using a 28 nm production process at TSMC. With a die size of 294 mm² and a transistor count of 3,540 million it is a medium-sized chip. GK104 supports DirectX 12 (Feature Level 11_0). For GPU compute applications, OpenCL version 3.0 and CUDA 3.0 can be used. It features 1536 shading units, 128 texture mapping units and 32 ROPs.
Further reading: GK104 Architecture Whitepaper
