NVIDIA's AD103 GPU uses the Ada Lovelace architecture and is made using a 5 nm production process at TSMC. With a die size of 379 mm² and a transistor count of 45,900 million it is a large chip. AD103 supports DirectX 12 Ultimate (Feature Level 12_2). For GPU compute applications, OpenCL version 3.0 and CUDA 8.9 can be used. Additionally, the DirectX 12 Ultimate capability guarantees support for hardware-raytracing, variable-rate shading and more, in upcoming video games. It features 10240 shading units, 320 texture mapping units and 112 ROPs. Also included are 320 tensor cores which help improve the speed of machine learning applications. The GPU also contains 80 raytracing acceleration cores.
