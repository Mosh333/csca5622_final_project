The Radeon Vega 3 Embedded is an integrated graphics solution by AMD, launched on September 6th, 2018. Built on the 14 nm process, and based on the Raven graphics processor, the device supports DirectX 12. This ensures that all modern games will run on Radeon Vega 3 Embedded. Unlike the fully unlocked Radeon RX Vega 11, which uses the same GPU but has all 704 shaders enabled, AMD has disabled some shading units on the Radeon Vega 3 Embedded to reach the product's target shader count. It features 192 shading units, 12 texture mapping units, and 4 ROPs. The GPU is operating at a frequency of 300 MHz, which can be boosted up to 1001 MHz.
Its power draw is rated at 15 W maximum.
