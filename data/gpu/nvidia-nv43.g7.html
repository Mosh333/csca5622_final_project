NVIDIA's NV43 GPU uses the Curie architecture and is made using a 110 nm production process at TSMC. With a die size of 154 mm² and a transistor count of 146 million it is a small chip. NV43 supports DirectX 9.0c (Feature Level 9_3). For GPU compute applications, OpenCL version N/A can be used. It features 8 pixel shaders and 3 vertex shaders, 8 texture mapping units and 4 ROPs. Due to the lack of unified shaders you will not be able to run recent games at all (which require unified shader/DX10+ support).
