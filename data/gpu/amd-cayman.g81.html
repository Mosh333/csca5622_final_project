AMD Cayman

Cayman
Block Diagram
VLIW Diagram
Information
Information 2
AMD's Cayman GPU uses the TeraScale 3 architecture and is made using a 40 nm production process at TSMC. With a die size of 389 mm² and a transistor count of 2,640 million it is a large chip. Cayman supports DirectX 11.2 (Feature Level 11_0). For GPU compute applications, OpenCL version 1.2 can be used. It features 1536 shading units, 96 texture mapping units and 32 ROPs.
