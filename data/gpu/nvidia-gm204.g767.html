NVIDIA's GM204 GPU uses the Maxwell 2.0 architecture and is made using a 28 nm production process at TSMC. With a die size of 398 mm² and a transistor count of 5,200 million it is a large chip. GM204 supports DirectX 12 (Feature Level 12_1). For GPU compute applications, OpenCL version 3.0 and CUDA 5.2 can be used. It features 2048 shading units, 128 texture mapping units and 64 ROPs.
