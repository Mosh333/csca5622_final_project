NVIDIA GK106S

GK106S
GK106S BGA
Die Shot
Block Diagram
SM Diagram
NVIDIA's GK106S GPU uses the Kepler architecture and is made using a 28 nm production process at TSMC. With a die size of 221 mm² and a transistor count of 2,540 million it is a medium-sized chip. GK106S supports DirectX 12 (Feature Level 11_0). For GPU compute applications, OpenCL version 3.0 and CUDA 3.0 can be used. It features 768 shading units, 64 texture mapping units and 16 ROPs.
