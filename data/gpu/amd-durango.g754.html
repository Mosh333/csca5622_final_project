AMD Durango

Durango
X871363-001
Die Shot
AMD's Durango GPU uses the GCN 1.0 architecture and is made using a 28 nm production process at TSMC. With a die size of 363 mm² and a transistor count of 5,000 million it is a large chip. Durango supports DirectX 11.2 (Feature Level 11_0). For GPU compute applications, OpenCL version 1.2 can be used. It features 896 shading units, 56 texture mapping units and 16 ROPs.
